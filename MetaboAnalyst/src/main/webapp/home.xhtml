Overview

Welcome to Ecologystics 1.0!

Ecologystics is a comprehensive platform designed for community ecology data analysis via a user-friendly, web-based interface. Inspired by the R-package Vegan, Ecologystics provides analysis tools for all scientists regardless of coding capabilities. Ecologystics facilitates the publication process by generating publication-quality figures, and its functionalities can be applied to non-ecological data types as well.
