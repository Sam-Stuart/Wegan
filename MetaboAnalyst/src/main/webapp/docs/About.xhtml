MetaboAnalyst Roadmap:

Version Key Feature Details
1.0 Statistics Focused on data processing, normalization and statistical analysis of metabolomics data;
2.0 Functions Incorporated metabolite set enrichment analysis (MSEA) and metabolic pathway analysis, as well as advanced two-factor and time-series analyses.
3.0 Performance Completed updated the underlying web framework for improved performance, with support for biomarker analysis, power analysis, and joint-pathway analysis.
4.0 Integration Completely revamped web interface, with three new modules (MS Peaks to Pathways, Biomarker Meta-analysis, and Network Explorer), together with a companion R package (MetaboAnalystR)

MetaboAnalyst is currently updated and maintained by Xia Lab at McGill Univeristy.

Implementation:

MetaboAnalyst was implemented using PrimeFaces library (v6.1) based on the JavaServer Faces Technology. The majority of the backend computations and visualization are carried out by over 600 functions written in R (v3.4.3). The communication between Java and R is established through TCP/IP using the Rserve program.
