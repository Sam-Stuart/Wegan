Data table: Plain text file (.csv or .txt)

Including supplemental data tables will enable more interesting and informative plots for certain analyses. Grouping data assigns categories to each sample to group one or more samples together. Constraining data assigns external conditions to each sample. One common example is environmental data. Note that constraining data is required for Co-Inertia Analysis and Analysis of Similarities.
