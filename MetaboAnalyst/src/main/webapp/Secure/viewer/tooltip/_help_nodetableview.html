The table shows detailed properties of all nodes in the current network. Double clicking on each column header to sort the table accordingly. The Expr. column shows the logFC values of the seed protein. When multiple datasets are uploaded (meta-analysis), the value is the number of times a given protein appears as significant in the individual data. A "-" indicates the corresponding node is not a seed protein.

You can search and highlight a single or multiple nodes within the network. Enter a key term to search; Use the checkboxes to select and highlight mulitple nodes. Click Delete button to delete selected nodes (note, the deleted nodes cannot be recovered).
