To identify enriched functions within the highlighted nodes/modules in the current network. You need to first highlight nodes or modules:
  • Module highlight: automatic: first perform module detection, then click on a module; manual:set to Module Mode (by clicking the corresponding icon on network toolbar), then double click a node on the netowrk to highlight the node together with their direct neighbours. Repeat the process to select more nodes.
  • Hub highlight: set to Single Mode (by clicking the corresponding icon on network toolbar), you can either select nodes from the node table on the left, or by double click a node directly from the network (tip: repeatedly clicking on the node will increase its size);
Note, you can double clicking the row of the result table to highlight the matched proteins in the module (don't forget to first set a color different from the module!).
