A dependent variable and an independent variable are correlated using a linear model. Predicted values are plotted against actual data, representing a measure of correlation.

Normality of residuals is one of the assumptions for a linear model. This plot (also called a 'Q-Q' plot) checks this assumption by comparing the distribution of model residuals (empirical) and theoretical normal distribution. In interpreting the plot, the closer points are to the diagonal line then the more likely the residuals are normally distributed.
