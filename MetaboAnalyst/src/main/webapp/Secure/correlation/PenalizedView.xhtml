Penalized Regression:

One dependent variable and one or more independent variables will be tested for correlation. The variables can be numeric or categorical. Note that all variables other than the dependent variable will be treated as independent variables. The penalized regression models will constrain the contribution of the independent variables on the dependent variable accordingly. The train data for model building is 70% of the dataset, while the test data for model testing is 30% of the dataset. Note that because the same penalty lambda (λ) will be applied equally to all coefficients, if you do not standardize your variables before running the regression the results may be influenced by differences in variable scale.
