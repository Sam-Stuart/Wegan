Missing value estimation:

Too many missing values (i.e. blanks or NAs) may cause difficulties for downstream analyses. There are several different methods to prevent future issues. The default method removes all variables with >50% missing values.

Ecologystics also offers methods to estimate missing values: replace by half of the minimum positive value of the original data, replace by the mean/median/minimum of the variable, k-nearest neighbour (KNN), probabilistic PCA (PPCA), Bayesian PCA (BPCA) method or Singular Value Decomposition (SVD) (ref.).
