This function implements Marti Anderson's PERMDISP2 procedure for the analysis of multivariate homogeneity of group dispersions (variances). Beta dispersal is a multivariate analogue of Leverne's test for homogeneity of variances. The 'Groups' feature allows the user to input a vector describing the group structure, values need to sum to the number of data points in the data set, and need to be delimited by comma's (i.e '8,14' with the Dune data set). The 'Labels' feature allows user to input labels for each of the groups (also delimited by comma's).
